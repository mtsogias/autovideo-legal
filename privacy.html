Privacy Policy

Last updated: February 2026

1. Overview

AutoVideo is a personal developer tool used exclusively by its creator. It is not a public application and has no end users other than the developer.

2. Data Collection

This application does not collect, store, or share any personal data from TikTok users. It only accesses the developer's own TikTok account via the TikTok Content Posting API to upload videos as drafts.

3. TikTok API Data

The application uses OAuth 2.0 to authenticate the developer's TikTok account. Access tokens are stored locally on the developer's machine and are never transmitted to third parties.

4. No Third-Party Sharing

No data is shared with third parties. No analytics, tracking, or advertising services are used.

5. Data Retention

No user data is retained. All processing happens locally on the developer's machine.

6. Contact

For privacy-related questions, contact: studio@amitamusic.de
